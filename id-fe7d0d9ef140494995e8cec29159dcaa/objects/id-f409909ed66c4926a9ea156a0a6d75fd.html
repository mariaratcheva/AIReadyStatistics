MCP Host is the AI applicatiion that manages one or more MCP clients MCP Client is a component that maintains connection to an MCP Server and obtains contaxt from an MCP Server for the MCP Host to use MCP Server is a program that provides context to MCP Clients. MCP Servers can execute locally or remotely Example: Claude Desktop has integrated MCP support. Chat App needs to support the protocols to run as an MCP Host 1. The MCP clients report the list of their connected tools and data sources 2. User asks: "What is our revenue for 2025?" 3. AI Agent determines that it needs the query tool and creates a query with parameter 2025 4. The MCP client catches the intent and creates a JSON request and it sends it to the MCP server 5. The MCP server returns the result to the MCP client in JSON format 6. The MCP client hands the result (the Context) to the AI Agent 7. The AI Agent creates the final result for the User
