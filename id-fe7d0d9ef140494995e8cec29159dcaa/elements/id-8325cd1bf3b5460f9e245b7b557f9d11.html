Data exposed with interpretative context ()
Principle. Data should be exposed together with the metadata needed to interpret, contextualise, and cite it correctly: concept measured, unit, reference period, population or territory covered, production status, source, and main methodological caveats. Rationale. An isolated value is not an interpretable answer. A human reader can rely on implicit context; an LLM cannot be assumed to share it and will otherwise fill gaps with plausible but statistically misleading assumptions.
