Prompts ()
Prompts are predefined templates or "shortcuts" provided by the server to help the user interact with the AI effectively. They often wrap complex instructions into a simple command. What they are: Reusable instructions that can take arguments. For example, a "Code Review" prompt might take a filename as an argument and then provide a specific set of instructions to the AI on how to audit that file. How they work: The server defines the prompt structure and the arguments it requires. When a user selects a prompt, the MCP server generates the full text for the LLM. User Value: They provide "Expertise-as-a-Service." Instead of a user having to write a perfect prompt for a specific task, the server provides a "Gold Standard" prompt designed by the developer of that service.
