Authoritative catalogues and discovery ()
Principle. For a given category of outputs there should be a single authoritative catalogue describing what is disseminated: the catalogue defines what exists, and search helps find items within it. Guidance. A domain may span several catalogues if their scope is clearly partitioned, but the same object should not be described in competing catalogues with divergent identifiers or status. Route-specific search implementations (website, API, chatbot, agent-facing) may differ in interface or ranking but must rely on coherent catalogues, metadata, and identifiers.
