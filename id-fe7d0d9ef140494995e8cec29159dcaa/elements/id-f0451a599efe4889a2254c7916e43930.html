One authoritative dissemination foundation ()
Principle. A single authoritative source of statistical outputs should feed all routes, website, APIs, agent-facing services, and any future access channels. Rationale. These routes are different exposure layers built on the same controlled dissemination assets. The access route may differ, but the underlying content must remain consistent across all of them.
